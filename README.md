# TrueYield - Real Profit Engine (Backend API)

TrueYield is a financial portfolio tracking engine built with **.NET 8**. The primary goal of this application is to solve a common investing problem: **comparing the foreign currency yield visually displayed by a broker against the actual, real yield calculated in the local currency (PLN)**.

By seamlessly integrating live stock prices from Finnhub with historical and current currency exchange rates from the National Bank of Poland (NBP), TrueYield reveals exactly how much of the profit was generated by the asset's growth versus how much was affected by currency exchange fluctuations.

> **Note on Frontend**: This repository focuses strictly on the Backend API. A React/Vite dashboard built to visually consume and display this cross-currency yield matrix is available in a separate repository (`TrueYield-Frontend`).

## 🚀 Quick Start Guide

This guide is designed to help you start the API locally in a few simple steps.

### Prerequisites
Before you begin, ensure you have the following installed on your machine:
1. **.NET 8 SDK**
2. **Docker Desktop** (⚠️ **Crucial:** Make sure the Docker Desktop application is actually open and running in the background before proceeding to Step 2).

### Step 0: Clone the Repository
If you haven't already, download the code to your local machine using Git:
```bash
git clone <YOUR_REPOSITORY_URL_HERE>
cd TrueYield-API
```

### Step 1: Obtain a Free Finnhub API Key
To fetch real-time market data, the backend requires a Finnhub API Key.
1. Go to [https://finnhub.io/register](https://finnhub.io/register) and create a free account.
2. In your Finnhub Dashboard, copy your **API Key** (a string of letters and numbers).
3. Open the following file in this repository:
   `TrueYield-API/appsettings.Development.json`
4. Replace `"YOUR_API_KEY_HERE"` with your actual Finnhub API key:
   ```json
   "Finnhub": {
     // Paste your Finnhub API Key below (e.g. "abc123yourkeyhere")
     "ApiKey": "abc123yourkeyhere"
   }
   ```
5. Save the file.

### Step 2: Start the PostgreSQL Database
TrueYield uses a Dockerized PostgreSQL database to securely store your portfolio positions.
1. Ensure your **Docker Desktop engine is turned on and running**.
2. Open a terminal (Command Prompt / PowerShell / Mac Terminal).
3. Navigate to the root directory of this backend repository (where the `docker-compose.yml` file is located).
4. Run the following command to start the database in the background:
   ```bash
   docker-compose up -d
   ```
*(Note: This also spins up pgAdmin on `http://localhost:5050` if you wish to inspect the database tables visually).*

### Step 3: Run the Backend (.NET API)
1. Keep the terminal open and navigate into the inner API folder:
   ```bash
   cd TrueYield-API
   ```
2. Start the backend application:
   ```bash
   dotnet run
   ```
3. Wait until you see `Now listening on: https://localhost:7150` in the terminal.

### Step 4: Explore and Test with Swagger
You do not strictly need the frontend repository to test the engine's capabilities!
Once the API is running, open your web browser and navigate to:
**[https://localhost:7150/swagger](https://localhost:7150/swagger)**

The Swagger UI provides a fully interactive interface where you can:
- **POST** `/api/portfolio`: Add new assets to your database.
- **GET** `/api/portfolio`: Calculate your cross-currency yield matrix.
- **GET** `/api/assets/search`: Test the in-memory blazing-fast asset autocomplete endpoint.
